
# PowerBoostCareer

PowerBoostCareer is an AI-powered platform designed to help users build market-aligned resumes, generate tailored cover letters, analyze ATS (Applicant Tracking System) scores, and launch their careers with a competitive edge. Built for job seekers, career changers, and professionals, it leverages Google Gemini AI to deliver actionable insights and automate key career tasks.

## Features

- **Resume Builder:** Create, tailor, and download professional resumes based on market analytics and project-driven achievements.
- **Cover Letter Generator:** Instantly generate personalized cover letters for any job application.
- **ATS Analysis:** Evaluate your resume against ATS criteria to maximize job search success.
- **Market Analytics:** Get real-time insights into job market trends, required skills, and role-specific competencies.
- **Job Search & Tracking:** Search for jobs, save listings, and track applications within the platform.
- **Project & Achievement Suggestions:** Receive AI-generated project ideas and achievement bullets to strengthen your profile.
- **Company Research & Interview Prep:** Access company research and generate interview questions tailored to your target role.

## How It Works

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Set up Gemini API:**
   Add your Gemini API key to `.env.local` as `GEMINI_API_KEY`.
3. **Run the app locally:**
   ```bash
   npm run dev
   ```

## Built With

- React & TypeScript
- Vite for fast development
- Google Gemini AI for content generation and analytics
